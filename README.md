# vue-vant-mobile

技术栈：vue全家桶、vantUI、axios
rem适配方案：postcss-pxtorem + amfe-flexible

## Project setup
```
npm i
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
